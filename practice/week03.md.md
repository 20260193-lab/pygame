**1. 기본 화면**

pygame 창 띄우고 원 하나 그리기, FPS 60으로 기본 루프 만들기



**2. 이동 추가**

wasd로 원 이동, 대각선 속도 보정



**3. 화면 제한**

원이 화면 밖으로 못 나가게 하기, 반지름 기준으로 경계 처리



**4. 플레이어 변경**

원 대신 삼각형으로 교체, 플레이어 중심 기준으로 그리기



**5. 장애물 원 생성**

랜덤 위치에 원 생성, 시간이 지나면 커지고 페이드 아웃



**6. 원 변경**

원을 내부가 비어있는 형태로 교체



**7. 충돌 시스템 \& 장애물 추가**

장애물에 닿으면 플레이어 삼각형 투명도가 흐려짐 ( 1번당 33%씩 ), 맞을 때 흔들림 효과 추가

화면 밖에서 빠르게 날라오는 사각형 장애물 추가



**8. 점수 시스템**

생존 시간 기반 점수 추가, 최고 점수 저장 및 불러오기



**9. 재시작 기능**

R키 누르면 게임 재시작 가능



**10. 밸런스 조정**

점수 오르는 양을 점점 더 많이 \& 장애물 난이도 증가
