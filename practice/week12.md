# R10unds 개발 메모

## 현재 게임 핵심 구조
- 10라운드 생존형 탄막 회피 게임
- 라운드가 올라갈수록:
  - 장애물 속도 증가
  - 장애물 생성 빈도 증가
  - 아레나 크기 감소
- 10라운드는 최종 페이즈 느낌으로 강화
- 플레이어는 대쉬 가능
- 힐 아이템 드랍 존재
- 피격 시:
  - 화면 흔들림
  - 짧은 감속
  - 무적 시간 적용
  - HP 잔상 UI 출력
